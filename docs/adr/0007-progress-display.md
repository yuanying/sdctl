# 0007. 生成中の進捗表示にプログレスバーを使用する

- Date: 2026-06-10
- Status: Accepted

## Context

AUTOMATIC1111 の API は `/sdapi/v1/progress` エンドポイントでポーリングにより進捗を取得できる。
画像生成には数秒〜数十秒かかるため、ユーザーへのフィードバックが必要。

## Decision

画像生成中はプログレスバーを表示し、ステップ数・パーセントをリアルタイムで更新する。

## Consequences

- ユーザーは生成の進捗を視覚的に確認でき、待機中の体験が改善される
- `/sdapi/v1/progress` へのポーリング実装が必要になる（適切なポーリング間隔の設定が必要）
- パイプや非 TTY 環境では進捗表示を抑制する考慮が必要
